<h3 id="ШҜШұШЁШ§ШұЩҮ">ШҜШұШЁШ§ШұЩҮ</h3>

ШҜШ§ЩҶШҙШ¬ЩҲЫҢ Ъ©Ш§ШұШҙЩҶШ§ШіЫҢ Ш§ШұШҙШҜ ШұШҙШӘЩҮвҖҢЫҢ ШӘШӯЩ„ЫҢЩ„ ШҜШ§ШҜЩҮ ЩҮШ§ ЩҲ ШҙШЁЫҢЩҮ ШіШ§ШІЫҢ ШіЫҢШіШӘЩ…вҖҢЩҮШ§ ЩҮШіШӘЩ….ШҙШҜЫҢШҜШ§ ШЁЩҮ ШӘШӯЩ„ЫҢЩ„ Ъ©Щ…ЫҢШҢ Ш§ЩӮШӘШөШ§ШҜ Ш§ЩҶШұЪҳЫҢ ЩҲ ЫҢШ§ШҜЪҜЫҢШұЫҢ Щ…Ш§ШҙЫҢЩҶ Ш№Щ„Ш§ЩӮЩ…ЩҶШҜ ЩҮШіШӘЩ… ЩҲ ШҜШ§ШұЩ… ШіШ№ЫҢ Щ…ЫҢЪ©ЩҶЩ… ШҜШұ Ш§ЫҢЩҶ ШӯЩҲШ¶ЩҮ ЩҫЫҢШҙШұЩҒШӘ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЩ….

ШҜШұ ШӯШ§Щ„ ШӯШ§Ш¶Шұ Щ…ШҙШәЩҲЩ„:

- ШӘЩӮЩҲЫҢШӘ ЩҮШұ ЪҶЩҮ ШЁЫҢШҙШӘШұ Щ…ЩҮШ§ШұШӘ ШҜШұ ЩҫШ§ЫҢШӘЩҲЩҶШҢ Ш§ЩӮШӘШөШ§ШҜШіЩҶШ¬ЫҢ ЩҲ ЫҢШ§ШҜЪҜЫҢШұЫҢ Щ…Ш§ШҙЫҢЩҶ
- Ш§ЩҶШ¬Ш§Щ… ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ЩҲШ§ЩӮШ№ЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ ШҜШ§ШҜЩҮ ЩҮШіШӘЩ…
- ШўЩ…Ш§ШҜЩҮвҖҢШіШ§ШІЫҢ ШЁШұШ§ЫҢ ШҙШұЩҲШ№ ЩҫЪҳЩҲЩҮШҙ Ш¬ШҜЫҢ ШҜШұ ШӯЩҲШ¶ЩҮ Ш§ЩӮШӘШөШ§ШҜ Ш§ЩҶШұЪҳЫҢ ШЁЩҮ Ш§ШіШӘШ§ШҜ ШұШ§ЩҮЩҶЩ…Ш§ЫҢЫҢ ШҜЪ©ШӘШұ Ш¬Щ„Ш§Щ„ЫҢ ЩҶШ§ШҰЫҢЩҶЫҢ ЩҮШіШӘЩ….

---

<h3 id="ЪҶШҙЩ… Ш§ЩҶШҜШ§ШІ">ЪҶШҙЩ… Ш§ЩҶШҜШ§ШІ</h3>

ЩҮШҜЩҒ Щ…ЩҶ ШұШіЫҢШҜЩҶ ШЁЩҮ Ш¬Ш§ЫҢЪҜШ§ЩҮ ЫҢЪ© **ШӘШӯЩ„ЫҢЩ„ЪҜШұ Ъ©Щ…Щ‘ЫҢ (Quant Analyst)** ШҜШұ ШӘЩӮШ§Ш·Ш№ Ш§ЩӮШӘШөШ§ШҜШіЩҶШ¬ЫҢШҢ ЫҢШ§ШҜЪҜЫҢШұЫҢ Щ…Ш§ШҙЫҢЩҶ ЩҲ ШЁШ§ШІШ§ШұЩҮШ§ЫҢ Ш§ЩҶШұЪҳЫҢ Ш§ШіШӘ вҖ” Ъ©ШіЫҢ Ъ©ЩҮ Щ…ШҜЩ„вҖҢЩҮШ§ЫҢ ШіШ§Ш®ШӘШ§ШұЫҢЩҗ Ш§ЩӮШӘШөШ§ШҜЫҢ ШұШ§ ШЁШ§ ШұЩҲШҙвҖҢЩҮШ§ЫҢ Щ…ШҜШұЩҶ ЫҢШ§ШҜЪҜЫҢШұЫҢ Щ…Ш§ШҙЫҢЩҶ ШӘШұЪ©ЫҢШЁ Щ…ЫҢвҖҢЪ©ЩҶШҜ ШӘШ§ ШӘШөЩ…ЫҢЩ…вҖҢЪҜЫҢШұЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ ШҜШ§ШҜЩҮ ШұШ§ ШҜШұ Щ…ШӯЫҢШ·вҖҢЩҮШ§ЫҢ ЩҲШ§ЩӮШ№ЫҢ ЩҲ ЩҫШұЩҶЩҲШіШ§ЩҶ Щ…Щ…Ъ©ЩҶ Ъ©ЩҶШҜ.

Щ…ШіЫҢШұ ЩҲШ§ЩӮШ№вҖҢЪҜШұШ§ЫҢШ§ЩҶЩҮвҖҢШ§ЫҢ Ъ©ЩҮ ШЁШұШ§ЫҢШҙ ШЁШұЩҶШ§Щ…ЩҮвҖҢШұЫҢШІЫҢ Щ…ЫҢвҖҢЪ©ЩҶЩ…:

- **Ъ©ЩҲШӘШ§ЩҮвҖҢЩ…ШҜШӘ (Ыұ-ЫІ ШіШ§Щ„):** ШӘШіЩ„Ш· ШЁШұ ЩҫШ§ЫҢЩҮвҖҢЩҮШ§ЫҢ ШұЫҢШ§Ш¶ЫҢШҢ ШўЩ…Ш§ШұЫҢ ЩҲ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢШӣ ШіШ§Ш®ШӘ ЪҶЩҶШҜ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ ЩҲШ§ЩӮШ№ЫҢ ЩӮШ§ШЁЩ„ Ш§ШұШ§ШҰЩҮ (ШӘШӯЩ„ЫҢЩ„ ШіШұЫҢ ШІЩ…Ш§ЩҶЫҢШҢ Щ…ШҜЩ„вҖҢШіШ§ШІЫҢ ЩӮЫҢЩ…ШӘ Ш§ЩҶШұЪҳЫҢШҢ ШЁЪ©вҖҢШӘШіШӘ Ш§ШіШӘШұШ§ШӘЪҳЫҢ)
- **Щ…ЫҢШ§ЩҶвҖҢЩ…ШҜШӘ (ЫІ-Ыі ШіШ§Щ„):** ЩҲШұЩҲШҜ ШЁЩҮ ЫҢЪ© ЩҶЩӮШҙ ШӘШӯЩ„ЫҢЩ„ЫҢ/ШӘШӯЩӮЫҢЩӮШ§ШӘЫҢ ШҜШұ ШӯЩҲШІЩҮвҖҢЫҢ Ш§ЩӮШӘШөШ§ШҜ Ш§ЩҶШұЪҳЫҢ ЫҢШ§ ШұЫҢШіЪ© Ъ©Щ…Щ‘ЫҢШҢ Щ…ШҙШ§ШұЪ©ШӘ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Щ…ШӘЩҶвҖҢШЁШ§ШІ Щ…ШұШӘШЁШ· ШЁШ§ Щ…ШҜЩ„вҖҢШіШ§ШІЫҢ Щ…Ш§Щ„ЫҢ ЩҲ Ш§ЩӮШӘШөШ§ШҜЫҢ
- **ШЁЩ„ЩҶШҜЩ…ШҜШӘ (Ыі-Ыө ШіШ§Щ„):** ШӘШЁШҜЫҢЩ„ ШҙШҜЩҶ ШЁЩҮ ЩҒШұШҜЫҢ Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЫҢЩҶ ШҜЩҶЫҢШ§ЫҢ ШӘШҰЩҲШұЫҢ Ш§ЩӮШӘШөШ§ШҜШіЩҶШ¬ЫҢ ЩҲ ШҜЩҶЫҢШ§ЫҢ Ш№Щ…Щ„ЫҢЩҗ ШҜШ§ШҜЩҮ ЩҲ ML ЩҫЩ„ ШЁШІЩҶШҜШӣ Ъ©Ш§Шұ ШҜШұ ЫҢЪ© ШҙШұЪ©ШӘ ШЁЫҢЩҶвҖҢШ§Щ„Щ…Щ„Щ„ЫҢШҢ ШҜШіЪ© Щ…Ш№Ш§Щ…Щ„Ш§ШӘЫҢ Ъ©Ш§Щ„Ш§ (Commodities)ШҢ ЫҢШ§ ЩҶЩҮШ§ШҜ ЩҫЪҳЩҲЩҮШҙЫҢ Щ…Ш№ШӘШЁШұ ШҜШұ ШӯЩҲШІЩҮвҖҢЫҢ Ш§ЩҶШұЪҳЫҢ

Ш§ЫҢЩҶ Щ…ШіЫҢШұ Ш¬Ш§ЩҮвҖҢШ·Щ„ШЁШ§ЩҶЩҮ Ш§ШіШӘ Ш§Щ…Ш§ ЪҜШ§Щ…вҖҢШЁЩҮвҖҢЪҜШ§Щ… ЩҲ ЩӮШ§ШЁЩ„ Ш§ЩҶШҜШ§ШІЩҮвҖҢЪҜЫҢШұЫҢ Ш·ШұШ§ШӯЫҢ ШҙШҜЩҮ вҖ” ЩҶЩҮ ЫҢЪ© ШұШӨЫҢШ§ЫҢ Ш§ЩҶШӘШІШ§Ш№ЫҢШҢ ШЁЩ„Ъ©ЩҮ ЫҢЪ© ЩҶЩӮШҙЩҮвҖҢЫҢ ШұШ§ЩҮ ШЁШ§ ЩҶЩӮШ§Ш· Ш№Ш·ЩҒ Щ…ШҙШ®Шө.

---

<h3 id="ШӘШӯШөЫҢЩ„Ш§ШӘ">ШӘШӯШөЫҢЩ„Ш§ШӘ</h3>

- **Ъ©Ш§ШұШҙЩҶШ§ШіЫҢ Ш§ШұШҙШҜ** ШӘШӯЩ„ЫҢЩ„ ШҜШ§ШҜЩҮ ЩҮШ§ ЩҲ Щ…ШҜЩ„ШіШ§ШІЫҢ ШіЫҢШіШӘЩ… ЩҮШ§
- **Ъ©Ш§ШұШҙЩҶШ§ШіЫҢ Ш§ШұШҙШҜ** ЩҒЫҢШІЫҢЪ© Щ…ЩҮШҜШіЫҢ ЪҜШұШ§ЫҢШҙ Ш§ЩҶШұЪҳЫҢ
